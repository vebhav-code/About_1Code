SYMPTOM Users report being logged out ~40 seconds after a successful login, seemingly at random. No error is shown client-side. Happens more on slower connections.

Here's the actual file. The bug (or bugs) are in here somewhere — read it the way you'd read a real PR.

1  class AuthService {
2    constructor() {
3      this.refreshTimer = null;
4    }
5
6    async login(email, password) {
7      const res = await fetch('/api/login', {
8        method: 'POST',
9        headers: { 'Content-Type': 'application/json' },
10       body: JSON.stringify({ email, password })
11     });
12     const data = await res.json();
13
14     localStorage.setItem('access_token', data.token);
15     localStorage.setItem('expires_in', data.expiresIn);
16     this.scheduleRefresh(data.expiresIn);
17     return data;
18   }
19
20   scheduleRefresh(expiresIn) {
21     // expiresIn arrives from the server in seconds
22     this.refreshTimer = setTimeout(() => {
23       this.refreshToken();
24     }, expiresIn); ← setTimeout expects ms, expiresIn is seconds
25   }
26
27   async refreshToken() {
28     const res = await fetch('/api/refresh', { method: 'POST' });
29     const data = await res.json();
30     localStorage.setItem('access_token', data.token);
31     this.scheduleRefresh(data.expiresIn);
32   }
33
34   logout() {
35     localStorage.removeItem('access_token');
36     localStorage.removeItem('expires_in');
37     // refreshTimer is never cleared here
38   } ← old timer keeps firing after logout
39 }

    // why this exists

    AI made code cheap.
    It made judgment expensive.

    Every junior dev now ships code an AI wrote. That code compiles, passes lint, and is often correct on the first try. The bottleneck moved — from typing the fix to knowing what's actually broken and telling the AI precisely enough to trust the output.

    1Code is built for that bottleneck. We never ask you to write working code. We ask you to think like the engineer who's read this exact bug before — and to prove it in the prompt.

    We never score syntax. Only diagnosis quality and prompt precision.

    02
