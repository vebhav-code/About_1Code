Traditional platforms test coding.
1Code tests how you think.

You solve realistic software bugs using AI, submit your AI conversation, and get rated on your debugging process — not your typing speed.

Real bugs, not puzzles

Every challenge is a realistic project with a real symptom — the kind you'd actually hit in production, not an algorithm riddle.

Your AI conversation is the artifact

Chat directly with the built-in AI assistant in your browser workspace. We evaluate your debugging transcript and code changes together.

Scored on process, not speed

Problem understanding, prompt quality, AI collaboration, and correctness — each rated separately so you know exactly what to sharpen.

// the workflow

From symptom to score in the in-browser workspace

Nine steps, zero file uploads. Scroll to walk through it.

01

Login / Register

Create your account and pick up where you left off — your rating persists per domain.

02

Choose a debugging mission

Browse challenges by domain — Auth, Database, Performance, Security — and pick your difficulty.

03

Read the scenario

Review the bug symptom and scenario — just like a real production ticket.

04

Start the challenge

Opens the in-browser workspace with the buggy code pre-loaded in an editor and a live AI chat panel alongside it.

05

Debug directly in the browser

Edit the code, ask the AI chat for help, both are saved automatically as you go.

06

Submit when ready

Submit when ready (or when the timer runs out).

07

AI evaluation

Gemini grades your session's real code and real chat transcript.

08

Score report

View your detailed score breakdown and overall evaluation.

09

Leaderboard

See where your score ranks against developers on the global leaderboard.
